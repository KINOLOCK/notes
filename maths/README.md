



## Asocijativnost

```
(a + b) + c =  
a + (b + c) = 
a + b + c
```

Zagrade nisu bitne.  
*Redoslijed operacija* nije bitan, može se paralelizovat.

## Komutativnost

`a + b = b + a`

*Redoslijed operanada* nije bitan.
Od francuske riječi "commuter" koja znači zamijeniti, obrnuti.   
Što će reći komutativno-flippable-obrtljivo.

## Distributivnost

`a(b + c) = ab + ac`
